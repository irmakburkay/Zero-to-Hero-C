> **From Zero to Hero in C Programming** - A comprehensive journey through C programming, data structures, algorithms, and neural networks from scratch.

## 🎯 Overview

This repository documents a complete learning journey in C programming, starting from basic concepts and culminating in a **fully functional neural network implementation**. While the initial goal was to solve the XOR problem, the project has evolved into a comprehensive C programming showcase.

## 📁 Project Structure

```
Zero-to-Hero-C/
├── 🧠 neural_network/     # Complete neural network from scratch
├── 📚 c101/               # C programming fundamentals
├── 🏗️  data_structures/    # Stack, Queue implementations
├── 🎯 exercies/           # Algorithm challenges & solutions
└── 📖 README.md           # This file
```

## ✨ Key Features

- **🧠 Neural Network**: Complete implementation with backpropagation
- **📊 Multiple Activation Functions**: Sigmoid, ReLU, Tanh
- **🎯 Various Test Cases**: XOR, parity, classification, regression
- **🏗️ Data Structures**: Stack and Queue implementations
- **🎮 Algorithm Challenges**: Real-world problem solutions

## 🚀 Quick Start

### Neural Network
```bash
cd neural_network
gcc -o nn *.c -lm
./nn
```


## 🎓 Learning Path

1. **C Fundamentals** (`c101/`) - Variables, arrays, pointers, memory management
2. **Data Structures** (`data_structures/`) - Stack and queue implementations
3. **Algorithms** (`exercies/`) - Problem-solving and optimization
4. **Neural Networks** (`neural_network/`) - AI from mathematical principles
